# Inputs

An input is the most basic element for inputting and editing data in an application. Different types of inputs are available for different types of input data to be used separately or combined together in a form. All form inputs has a short label describing the type of the expected data and should be positioned into a fieldset in a grid.

## Switch

The Switch component should be used to toggle on or off a limited number of choices.

## TextInput

The Text Input component is used for inputting text. It has several states depending on what input data is required and if the input fields has any requriements.

<Canvas>
  <TextInput placeholder="Username" type="text" />
  <TextInput placeholder="Password" type="password" />
  <TextInput
    placeholder="Field with criteria"
    criteria={[
      {
        criteria: 'Longer than 6 characters',
        fulfilled: false,
        condition: { type: 'length', rule: 6 }
      },
      {
        criteria: 'Contains at least 1 lowercase letter',
        fulfilled: true,
        condition: { type: 'regex', rule: new RegExp('^(?=.*?[a-z]).{1,}$') }
      },
      {
        criteria: 'Contains at least 1 uppercase letter',
        fulfilled: true,
        condition: { type: 'regex', rule: new RegExp('^(?=.*?[A-Z]).{1,}$') }
      },
      {
        criteria: 'Contains at least 1 number',
        fulfilled: true,
        condition: { type: 'regex', rule: new RegExp('^(?=.*?[0-9]).{1,}$') }
      },
      {
        criteria: 'Contains at least 1 of %, &, /, ! or ?',
        fulfilled: true,
        condition: { type: 'regex', rule: new RegExp('^(?=.*?[!%&/?]).{1,}$') }
      }
    ]}
  />
</Canvas>
